# STM32 FreeRTOS LED Controller

## 📌 Overview
This project demonstrates multitasking using FreeRTOS on STM32F407.

## 🧠 Features
- Multiple LED control tasks
- Button interrupt handling
- Task priority management

## 🛠 Hardware Used
- STM32F407
- On-board LEDs
- Push button

## 💻 Software Used
- STM32CubeIDE
- Embedded C
- FreeRTOS

## 🔄 RTOS Concepts
- Task scheduling
- vTaskDelay
- Priority-based execution
